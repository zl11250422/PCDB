## Environment

[PyTorch >= 1.7](https://pytorch.org/)

## How To Test

```
python test.py -opt options/test/benchmark_BSRN_x4.yml
```
The testing results will be saved in the ./results folder.

## How To Train

```
python train.py -opt options/train/train_BSRN_x4.yml
```
